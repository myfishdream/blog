## 基于VitePress的博客文档

这是一个使用VitePress构建的现代化博客文档系统。VitePress作为一个高效的静态站点生成器(SSG)，专为创建快速、以内容为中心的网站而设计

核心优势在于能够将Markdown格式的内容转换为美观、响应式的HTML页面，让您专注于内容创作而非技术细节。

[VitePress官网](https://vitepress.dev/zh/)
